# 脚本是怎么运行的

## 一、运行脚本的基本流程

```
写代码 → 保存为文件 → 运行 → 得到结果
```

就这么简单，三步。

---

## 二、具体操作（以 Python 为例）

### 第 1 步：安装 Python

- 去官网下载：https://python.org
- 安装时勾选 "Add Python to PATH"
- 验证安装成功：打开终端输入 `python --version`

### 第 2 步：写一个脚本文件

创建一个文件，比如叫 `hello.py`，内容就一行：

```python
print("Hello World")
```

### 第 3 步：运行它

打开终端，输入：

```bash
python hello.py
```

你会看到输出：

```
Hello World
```

**这就是运行一个脚本的全过程。**

---

## 三、脚本是怎么"干活"的

脚本的本质就是：**按照你写的步骤，自动执行操作。**

### 例子：批量重命名文件

不用脚本你要做的：
1. 打开文件夹
2. 点第一个文件
3. 按 F2 改名
4. 回车
5. 重复 100 次

用脚本：
```python
import os

folder = "C:/my-files"
for i, filename in enumerate(os.listdir(folder)):
    new_name = f"文件_{i+1}.txt"
    os.rename(f"{folder}/{filename}", f"{folder}/{new_name}")
```

运行一下，100 个文件全部改好。

## 四、你可能关心的问题

### 脚本需要安装什么吗？
- Python 本身需要安装一次
- 用到的库（比如处理 Excel 的 openpyxl）需要用 `pip install` 安装
- 之后就不需要再装了

### 脚本运行在什么地方？
- 就在你自己的电脑上运行
- 不需要服务器
- 不需要联网（除非脚本本身需要联网）

### 脚本安全吗？
- 脚本只是在你电脑上执行操作
- 只要不运行来历不明的脚本就没问题
- 用 Claude Code 帮你写的代码，你可以让 Claude 解释每一行在做什么

### 脚本怎么给别人用？
- 把 .py 文件发给对方
- 告诉对方安装 Python
- 对方运行 `python 文件名.py` 即可
- 也可以打包成 .exe，对方双击就能运行（不需要装 Python）
