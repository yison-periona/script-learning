# 脚本学习路线图

## 第 1 阶段：认识脚本（你在这里 ✅）

- 知道脚本是什么
- 知道脚本怎么运行
- 知道脚本可以卖钱

## 第 2 阶段：做一个最简单的脚本

- 用 Claude Code 写一个简单脚本
- 比如：批量重命名文件、文件夹统计
- 自己运行一次，感受效果
- 时间：30 分钟

## 第 3 阶段：做 3~5 个不同类型的脚本

- 文件处理类（批量重命名、文件分类）
- 数据处理类（Excel 合并、CSV 清洗）
- 自动化类（网页信息提取、自动发邮件）

每做完一个：
- 保存代码
- 截一张效果图
- 记录用了什么方法

## 第 4 阶段：放到作品集里

- 把做过的脚本传到 GitHub
- 在你的个人主页上展示
- 每个脚本写清楚：做什么的、效果是什么

## 第 5 阶段：开始接单

- 在闲鱼/小红书发一条接单信息
- 说明你能做什么类型的脚本
- 附上作品截图或演示
- 从最简单的需求开始接

## 第 6 阶段：扩大

- 积累模板，提高交付速度
- 尝试打包成 .exe 卖
- 拓展到更多需求类型
- 提高价格
