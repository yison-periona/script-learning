# 脚本变现的思路

## 一、谁来买你的脚本

### 1. 不懂技术的普通人
- 想批量处理文件但不会
- 想整理数据但不会 Excel 高级操作
- 想从网上批量下载图片但不会

### 2. 没时间的技术人员
- 知道怎么做但不想花时间
- 愿意花小钱省时间

### 3. 学生群体
- 需要整理课程资料
- 需要处理实验数据
- 需要批量生成文档

---

## 二、脚本怎么卖

### 方式一：接定制单
- 客户告诉你他要什么
- 你用 Claude Code 写出来
- 交付脚本文件
- 收钱

**适合起步**，因为需求明确，容易完成。

### 方式二：做通用工具卖多次
- 你做一个通用脚本（比如"Excel 合并工具"）
- 放到闲鱼/淘宝上卖
- 多个人买，同一个东西卖多次

**适合后期**，做一次卖多次。

### 方式三：打包成小软件
- 给脚本加一个简单的界面
- 打包成 .exe
- 别人双击就能用，不需要装 Python

**看起来更专业**，但需要多花一点时间。

## 三、定价参考

| 脚本复杂度 | 参考价格 | 举例 |
|-----------|---------|------|
| 非常简单 | 30~50 元 | 批量重命名、文件分类 |
| 简单 | 50~150 元 | Excel 合并、数据清洗 |
| 中等 | 150~500 元 | 网页信息提取、自动生成报告 |
| 复杂 | 500~2000 元 | 完整的数据处理流程 |

## 四、在哪里接单

- **闲鱼**：搜索"Python 脚本"、"数据处理"看看别人怎么卖
- **小红书**：发你做过的脚本效果展示
- **同学/朋友圈**：最直接的渠道
- **程序员客栈**：可以注册看看
- **Upwork / Fiverr**：海外平台，价格更高但有语言门槛
