# 脚本用什么语言写

## 最推荐的：Python

对学生来说，Python 是写脚本最好的选择。

### 为什么选 Python

- 语法最接近人类语言，容易看懂
- Claude Code 最擅长帮你写 Python
- 有海量现成的库（别人写好的工具包），不用自己从零造轮子
- 几乎所有脚本需求都能用 Python 搞定

### Python 能做什么脚本

| 需求 | 用到的库 |
|------|---------|
| 处理 Excel/CSV | openpyxl, pandas |
| 处理 PDF | PyPDF2, pdfplumber |
| 操作文件/文件夹 | os, shutil（Python 自带） |
| 网页信息提取 | requests, BeautifulSoup |
| 发送邮件 | smtplib（Python 自带） |
| 图片处理 | Pillow |
| 自动生成文档 | python-docx |

### 不需要记住这些库

你不需要背任何库的名字。

你只需要告诉 Claude Code：
> "帮我写一个脚本，把 50 个 Excel 合并成一个"

Claude Code 会自动选择合适的库并帮你写代码。

---

## 其他可选语言

### JavaScript / Node.js
- 适合做网页相关的脚本
- 比如自动化操作浏览器

### Shell / Bash
- 适合系统管理类的操作
- 比如 Linux/Mac 上的文件管理

### 对初学者来说
**只用 Python 就够了。**
